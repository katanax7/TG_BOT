# TG_BOT — Phoenix 2

Телеграм-бот с системой регистрации пользователей с помощью FSM `aiogram`.

## Возможности

- Регистрация с выбором роли
- FSM логика регистрации (aiogram 3.x)
- Подготовка к добавлению модерации и ролей админов
